# TrackView(仿知乎可拖动悬停按钮)
![0](https://github.com/XiaogegeChen/TrackView/blob/master/screenshot/device-2019-05-11-203831.gif)
## 快速使用
在工程根目录的build.gradle中添加依赖
```maven { url 'https://jitpack.io' }```<br>如下
```allprojects {
    repositories {
        google()
        jcenter()
        maven { url 'https://jitpack.io' }
    }
}```<br>
